Class: CustomSlider

CustomSlider

This is an implementation of slider plugin. The plugin is capable of executing user defined horizontal sliders such as Simple-Slider, Range-slider, Fixed-Range-Slider and Draggable-range-Slider. It also has tool-tip, label and tick functionalities. This plugin is forked and extended from bootstrap-slider [https://github.com/seiyria/bootstrap-slider authored by Kyle Kemp and Rohit Kalkur].
